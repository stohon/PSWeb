>_Welcome to PowerShell Console
PowerShell Console allows for remote execution of predefined PowerShell scripts. This solution was designed to allow scripts to be run locally and in this console without modifications.
To learn more see the example scripts and help files listed below and select them from the File menu drop down.
PowerShell Console Directory
1. invoke-webrequest.ps1:
Returns a web page based on URL provided.
2. write-in.ps1:
Pass in mulitple variables which are then returned. Illistrates how to use the write-in function to create scripts.
